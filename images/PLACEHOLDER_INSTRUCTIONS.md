# Instrucciones para Mockups de Proyectos

## Nombres de archivos requeridos:

1. **proyecto1-desktop.png** - Portfolio profesional para diseñador web
2. **proyecto2-desktop.png** - Sistema de turnos para barberías
3. **proyecto3-desktop.png** - Web para artista textil (Casi como ella)

## Recomendaciones técnicas:

### Especificaciones de imagen:
- **Formato:** PNG o JPG (preferentemente PNG con fondo transparente)
- **Tamaño recomendado:** 800-1000px de ancho (la altura se adapta automáticamente)
- **Resolución:** 72 DPI mínimo (hasta 300 DPI para máxima claridad)
- **Proporción:** 16:9 o similar (se verá mejor)
- **Color:** Fondos oscuros preferentemente para mantener consistencia

### Consejos de diseño:
- Captura el sitio completo o una sección representativa
- Incluye desktop y/o mobile en la misma imagen si es posible
- Usa fondos oscuros o con alto contraste
- Mantén un estilo consistente con la paleta de Axus

### Cómo reemplazar:
1. Exporta tus mockups como PNG o JPG
2. Copia los archivos a la carpeta `/images/` de este proyecto
3. Asegúrate de usar estos nombres exactamente:
   - `proyecto1-desktop.png`
   - `proyecto2-desktop.png`
   - `proyecto3-desktop.png`

### Efectos que se aplican automáticamente:
✓ Sombra suave con glow cyan  
✓ Animación de flotación (float)  
✓ Rotación sutil (±1 grado)  
✓ Efecto hover con scale(1.02) y sombra profunda  
✓ Bordes redondeados (20px)  
✓ Responsive: se adapta a todos los tamaños de pantalla

---

**Nota:** Los placeholders SVG se reemplazarán automáticamente cuando subas las imágenes con los nombres correctos.
